# GeoRush

Et enkelt Geometry Dash-inspirert spill for mobil (horisontalt).

## Spill

- **Trykk / Space** for å hoppe
- Hold inne for å hoppe igjen når du lander
- Unngå spikes, nå portalen

## Brett

1. First Steps  
2. Spike Alley  
3. Sky Blocks  
4. Neon Night  
5. Final Rush  

## Lokalt

Åpne `index.html` i nettleseren, eller:

```bash
python3 -m http.server 8080
```

## GitHub Pages

Publiseres automatisk fra `main`-branchen.
