# GeoRush

Et enkelt Geometry Dash-inspirert spill for mobil (horisontalt).

## Spill

- **Trykk / Space** for å hoppe
- Hold inne for å hoppe igjen når du lander
- Samle **coins** (teller øverst til høyre)
- Kjør inn i **portaler** for coin-soner — du kommer ut samme sted
- Unngå spikes, sager, lasere og tak-spikes

## Brett

1. First Steps  
2. Spike Alley  
3. Sky Blocks  
4. Neon Night  
5. **Flash Rush** (~2 min, flashing lights, portaler)  
6. Final Rush  

## Lokalt

Åpne `index.html` i nettleseren, eller:

```bash
python3 -m http.server 8080
```
